## Estruturas Implementadas

As seguintes estruturas de dados foram implementadas neste repositório:

1. **Lista Simplesmente Ligada (Singly Linked List)**: Implementação básica de uma lista encadeada.
2. **Pilha (Stack)**: Estrutura de dados do tipo LIFO (Last In, First Out).
3. **Fila (Queue)**: Estrutura de dados do tipo FIFO (First In, First Out).
4. **Árvore Binária de Busca (Binary Search Tree - BST)**: Estrutura de árvore para armazenar dados de forma ordenada.
5. **Grafo (Graph)**: Implementação de grafos usando lista de adjacência.
6. **Tabela de Hash (Hash Table)**: Estrutura de dados que utiliza uma função de hash para armazenar dados.

## Requisitos

Para rodar o projeto, é necessário ter os seguintes requisitos instalados:

- **Compilador C** (ex.: GCC ou Clang)
- **Sistema Operacional**: Linux, macOS ou Windows com suporte a compilação em C

## Como Usar

1. Clone o repositório para seu ambiente local:

```bash 
git clone https://github.com/seuusuario/projeto-estruturas-de-dados.git
```

2. Navegue até o diretório do projeto:
```bash
cd projeto-estruturas-de-dados
```
3. Compile o código utilizando o compilador de sua escolha. Exemplo com GCC:
```bash
gcc -o main main.c
```
4. Execute o programa:
```bash
./main
```
