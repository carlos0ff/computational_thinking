## 🔍 Introdução
Este repositório é dedicado às atividades da disciplina de **Estrutura de Dados I e II**, ministrada pelo Professor Edkallen. <br/>
Ele contém implementações de várias estruturas de dados fundamentais e os projetos práticos realizados durante o curso. 📖✨

---

## 🛠️ Estruturas Implementadas

As seguintes estruturas de dados foram implementadas neste repositório:

1. **Lista Simplesmente Ligada (Singly Linked List)**: Implementação básica de uma lista encadeada. 🔗
2. **Pilha (Stack)**: Estrutura de dados do tipo LIFO (Last In, First Out). 📦
3. **Fila (Queue)**: Estrutura de dados do tipo FIFO (First In, First Out). 🕒
4. **Árvore Binária de Busca (Binary Search Tree - BST)**: Estrutura de árvore binária usada para armazenar dados de forma ordenada. 🌳
5. **Grafo (Graph)**: Implementação de grafos usando lista de adjacência, representando relações entre objetos. 🌐
6. **Tabela de Hash (Hash Table)**: Estrutura de dados que utiliza uma função de hash para mapear dados de maneira eficiente. 🔑

---

## ⚙️ Requisitos

Para rodar o projeto, você precisará dos seguintes requisitos:

- **Compilador C**: GCC, Clang ou qualquer outro compilador C. 🖥️
- **Sistema Operacional**: Funciona em Linux, macOS ou Windows com suporte a compilação C. 🏞️

## 🚀 Como Usar

1. **Clone o repositório** para o seu ambiente local:

    ```bash
    git clone https://github.com/seuusuario/projeto-estruturas-de-dados.git
    ```

2. **Navegue até o diretório** do projeto:

    ```bash
    cd projeto-estruturas-de-dados
    ```

3. **Compile o código** usando o compilador de sua escolha. Exemplo com o GCC:

    ```bash
    gcc -o main main.c
    ```

4. **Execute o programa**:

    ```bash
    ./main
    ```

## 📝 Lista de Exercícios

Neste repositório, você encontrará uma lista de **exercícios práticos** desenvolvidos durante o curso. Esses exercícios cobrem as implementações das estruturas de dados mencionadas anteriormente e ajudam a reforçar os conceitos aprendidos na disciplina. Todos os exercícios estão localizados na pasta [`exercises/`](./exercises/). 📑✍️

- **Exercício 1**: Implementação de uma **Lista Simplesmente Ligada** com operações básicas (inserção, remoção e busca).
- **Exercício 2**: Criação de uma **Pilha** utilizando lista encadeada e suas operações (push, pop, peek).
- **Exercício 3**: Implementação de uma **Fila** com operações de enfileirar e desenfileirar.
- **Exercício 4**: Desenvolvimento de uma **Árvore Binária de Busca** e suas operações principais (inserção, busca e remoção).
- **Exercício 5**: Implementação de um **Grafo** utilizando lista de adjacência, com as operações de inserção e remoção de arestas.
- **Exercício 6**: Criação de uma **Tabela de Hash** para armazenar dados de maneira eficiente.

Esses exercícios são essenciais para o entendimento das principais **estruturas de dados** e suas aplicações práticas. 💡
